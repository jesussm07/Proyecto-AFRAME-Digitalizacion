# Museo virtual en A‑Frame

Este proyecto es una escena de realidad virtual creada con A‑Frame que representa un pequeño museo virtual con 3 cuadros famosos colocados en una galería. El usuario puede recorrer el espacio y observar cada obra desde diferentes puntos de vista.

## Tecnologías utilizadas

- [A‑Frame](https://aframe.io/) para construir la escena 3D y la experiencia de realidad virtual.
- HTML y CSS básicos para la estructura y el estilo de la página.
- Extensión Live Server Preview compatible con WebVR/WebXR para visualizar la escena.

## Contenido de la escena

En el museo virtual se muestran tres cuadros famosos colocados en la pared principal de la galería.  
Cada cuadro está sobre su propio marco y separado del resto para que se puedan ver de forma individual.

- Cuadro 1: La noche estrellada – Vincent van Gogh.
- Cuadro 2: El grito – Edvard Munch.
- Cuadro 3: La Mona Lisa – Leonardo da Vinci.

## Cómo ejecutar el proyecto
1. Descarga este repositorio en tu ordenador.  
2. Abre el archivo `index.html` dentro de la carpeta indice con un navegador compatible (por ejemplo, Chrome o Firefox). 
3. Mueve la cámara con el ratón o con las teclas de movimiento para recorrer el museo virtual.
